動作環境
Windows10, 8, 8.1, 7
DirectX
メモリ512MB以上
Windows10、Windows7で動作確認。

起動方法
Shu.exe - ショートカットを起動してください。
(./Petit Shoot/Petit Shoot/Shu.exeが起動します。)

操作方法

キーボード：
上下左右キーで移動
spaceキーで決定、弾発射(押しっぱなしで連射)

ジョイパッド：
上下左右キーで移動
0ボタンで決定、弾発射(押しっぱなしで連射)

コンティニュー：
タイトル画面で上下キーで
クリアしたステージをセレクトできます。(spaceで決定)
(クリアステージ、ハイスコアはsave.binに保存されます。)
(今回のバージョンでははじめから全ステージ、エンディング(ステージ6)が見られます。)

また、中ボスのあるステージは中ボスを倒すと、
コンティニュー時に中間ポイントから復帰できます。
(中間ポイントはsave.binにセーブされません。)
